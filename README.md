# Number Statistics Calculator (Python)

This Python program lets the user enter integers one by one and then calculates basic statistics for the numbers entered.  
It displays how many numbers were entered, the sum, the average, the smallest number, and the biggest number.

## Features
- Accepts multiple integer inputs from the user
- Keeps track of how many numbers were entered
- Calculates the total sum
- Calculates the average
- Finds the smallest number
- Finds the biggest number

## Example Run

Welcome! Enter numbers one by one.  
Enter an integer: 5  
Enter another? (y/n) y  
Enter an integer: 10  
Enter another? (y/n) y  
Enter an integer: 3  
Enter another? (y/n) n  

3 numbers were entered.  
The sum is 18  
The average is 6.0  
3 was the smallest number entered.  
10 was the biggest number entered.
